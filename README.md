# Options Trading Simulator

This is a complete Vite + React project ready for GitHub and Vercel.

## Local run

1. Install Node.js
2. Open a terminal in this folder
3. Run:

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Create a new GitHub repository
2. Upload all files from this folder to the repository
3. In Vercel, choose **Add New Project**
4. Import the GitHub repository
5. Click **Deploy**

Vercel should detect this as a Vite app automatically.

## Notes

- Sessions are currently stored in the browser using local storage.
- The written rationale is assumed to be submitted separately on SurreyLearn.
- The app lets students save multiple sessions and select the best four, with at least two scenarios required across the selected four.
